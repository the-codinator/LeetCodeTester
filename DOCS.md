# Maven

Include the following dependency in your pom

```xml
<dependency>
  <groupId>org.codi</groupId>
  <artifactId>leetcode-tester</artifactId>
  <version>0.0.1</version>
  <!-- checkout latest version under releases -->
</dependency>
```
